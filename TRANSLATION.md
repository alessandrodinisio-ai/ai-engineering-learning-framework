# Translation Note

This project was translated from the Chinese fork [ai-engineering-from-scratch-zh](https://github.com/fancyboi999/ai-engineering-from-scratch-zh), which adapted the original [AI Engineering from Scratch](https://github.com/rohitg00/ai-engineering-from-scratch) curriculum into Simplified Chinese. The translation back to English preserves all code, URLs, file paths, and structural elements while converting Chinese prose into clear, concise English. We credit the Chinese fork maintainers for their work in adapting and expanding the curriculum. Contributors should consult the bilingual terminology table below when reviewing translated content or submitting changes that touch lesson documents.

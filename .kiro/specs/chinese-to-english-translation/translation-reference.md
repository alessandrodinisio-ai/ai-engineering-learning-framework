## Terms to Preserve in English (Do Not Translate)

The following terms should remain in English in the translated output. They are never translated:

`agent`, `token`, `prompt`, `transformer`, `LLM`, `MCP`, `API`, `GPU`, `CPU`, `PyTorch`, `NumPy`, `uv`, `pnpm`, `cargo`, `Claude`, `Cursor`, `Codex`, `Docker`, `vLLM`, `LangGraph`

All product names, library names, and tool names are preserved as-is.

## Non-Translatable Elements (Preserve Byte-Identical)

- Fenced code blocks (content between ` ``` ` markers)
- Inline code spans (text in single backticks)
- URLs (http/https links including query params and fragments)
- File paths (e.g., `phases/00-.../docs/zh.md`)
- Mermaid diagram blocks (` ```mermaid ` to ` ``` `)
- HTML tags and alignment blocks
- Badge image URLs
- ASCII art dividers (░▒ lines)
- Emoji status characters (✅, 🚧, ⬚)
- Table delimiter rows (pipe-and-dash lines)
- Frontmatter blocks (--- delimiters)
- Image references (`![alt](path)`)

---

## Translation Quality Rules

1. Direct, concise English — no filler phrases ("It is worth noting that", "at this point in time", "In order to")
2. Vary sentence length and rhythm
3. No promotional or hyperbolic language
4. Preserve paragraph count, bullet order, and section order
5. Average sentence length ≤ 20 words
6. Trust the reader — no over-explanation
